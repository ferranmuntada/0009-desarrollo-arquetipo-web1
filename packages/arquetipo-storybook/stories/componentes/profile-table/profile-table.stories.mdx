Componente:

Propiedades:
- title (string): Título de la caja.

- Para incorporar el component, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<mln-profile-table [title]="title">
  <!--
  Contenido de la caja:
    - Aquí se puede añadir una plantilla HTML. 
   -->
</mln-profile-table>
```
