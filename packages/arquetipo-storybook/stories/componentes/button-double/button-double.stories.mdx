Todos los tipos de componentes botón aceptarán los siguientes parámetros:

- active: (Boolean) Indica si el botón está activo o no.
- leftText: (String) Texto en la zona izquierda del botón.
- rightText: (String) Texto en la zona derecha del botón.
- actionBigButton: (EventEmitter) Emite un evento al hacer clic al botón.

## Button Double Active

Botón por defecto que acepta las propiedades:

### Ejemplo de uso:

Definición de modelo de datos del componente:

```typescript live=true
const active = true; 
const leftText = 'PIC'; 
const rightText = 'Botón';

const actionBigButton = (event: Event) => {};
```

- Para incorporar el component, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<mln-button-double
  [active]="active"
  [leftText]="leftText"
  [rightText]="rightText"
  (actionBigButton)="actionBigButton($event)"
></mln-button-double>
```
