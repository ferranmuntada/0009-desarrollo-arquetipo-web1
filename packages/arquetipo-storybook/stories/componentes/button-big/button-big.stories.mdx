- BigButtonModule

```js live=true
import { BigButtonModule } from "arquetipo-storybook/dist";
```

Todos los tipos de componentes botón aceptarán los siguientes parámetros:

- active: (Boolean) Indica si el botón está activo o no.
- text: (String) Texto que contendrá el botón.
- icon: (String) Icono que mostrará el botón, los iconos puede ser uno de los que se encuentran en la siguieten url. (https://material.io/resources/icons/?style=baseline). En caso de no indicar nada el botón mostrará un icono por defecto.
- callback (Subject) Observable que al que se puede subscribir.

### Ejemplo de uso de propiedades del componente.
```typescript live=true
const active = true; 
const icon = 'arrow_circle_up';
const text = 'Botón';

// Este callback es un observable al cual nos tenemos que suscribir para recibir para recibir la acción del botón (clic)
const callback = new Subject<boolean>();
callback.subscribe(() => console.log('Reacción del botón'));
```

- Para incorporar el component, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<mln-button-big [active]="active" [icon]="icon" [text]="text" [callback]="callback"></mln-button-big>
```
