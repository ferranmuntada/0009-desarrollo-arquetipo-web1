# COMPONENTE SELECT MENU

Modulo:

- SelectModule

```js live=true
import { SelectModule } from "arquetipo-storybook/dist";
```

Componente:

- Para incorporar el component, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<mln-select [settings]="settings" [listItem]="listItem" (actionSelected)="actionSelected($event)"></mln-select>
```

Componente custom selector. Acepta los siguiente par치metros:

- listItem: (Array) Vector con los elementos que contendr치 el custom selector. Se especifica en formato JSON y contiene los siguientes par치metros:
```typescript live=true
const listItem = [
    {
      name: 'nombre 1',
      icon: 'edit',
      id: 1,
      disabled: true
    },
    {
      name: 'nombre 2',
      icon: '',
      id: 2
    },
    {
      name: 'nombre 3',
      icon: '',
      id: 3
    },
    {
      name: 'nombre 4',
      icon: '',
      id: 4
    }
];
```
- settings: (Object) Datos a enviar para las propiedad del select.
```typescript live=true
const settings = {
      firstOption: "Ninguna",
      title: "Ver todos",
};
```
- actionSelected: (EventEmitter) Valor que devolver치 al seleccionar un elemento.
```typescript live=true
const actionSelected = (event: Event) => {};
```
