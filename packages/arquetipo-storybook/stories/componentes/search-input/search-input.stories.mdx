# COMPONENTE SEARCH INPUT

Modulo:

- SearchInputModule

```js live=true
import { SearchInputModule } from "arquetipo-storybook/dist";
```

Componente:

- Para incorporar el component, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<mln-search-input></mln-search-input>
```

Componente input para realizar búsquedas sobre una tabla. Acepta los siguiente parámetros:

- settings:
  - textInputDefault: (String) Texto que se quiere añadir por defecto, en caso de viajar vacío se pondrá el placeholder.
  - icon: (String) Acepta cualquiera de los inconos de la librería de material. https://material.io/resources/icons/
- amount: (String) Número de elementos encontrados para el cliente.
- placeholder: (String) Texto que figurará en el placeholder
- changes: (Action) Funcion a la que se quiere llamar una vez se introduzca el texto.

## Ejemplo de uso

Definición de modelo de datos del componente:

```js live=true
const settings = {
      textInputDefault: "Search",
      icon: "search",
    };
const amount = 5;
const placeholder =  "PlaceHolderText";
const changes = action("changes");
}
```

Declaración HTML:

```html live=true
<mln-search-input [settings]="settings" [amount]="amount" [placeholder]="placeholder" (changes)="changes"></mln-search-input>
```
