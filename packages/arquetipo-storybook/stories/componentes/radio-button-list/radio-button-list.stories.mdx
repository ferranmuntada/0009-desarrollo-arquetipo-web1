Modulo:

- RadioButtonListModule

```js live=true
import { RadioButtonListModule } from "arquetipo-storybook/dist";
```

Componente:
- list (Array):
```typescript live=true
const list = [
    { text: 'texto 1', checked: true },
    { text: 'texto 2' },
    { text: 'texto 3' },
    { text: 'texto 4' },
];
```
- actionRadioButton (output):
```typescript live=true
const actionRadioButton = (event: Event) => {}
```
- textLabel (string): Texto que aparece en la parte superior del radio button.
- vertical (boolean): Radio button en horizontal o vertical
- Para incorporar el component, el prefijo a utilizar para la etiqueta HTML es:
```html live=true
<mln-radio-button-list [list]="list" (actionRadioButton)="actionRadioButton($event)" textLabel="texto"></mln-radio-button-list>
```
