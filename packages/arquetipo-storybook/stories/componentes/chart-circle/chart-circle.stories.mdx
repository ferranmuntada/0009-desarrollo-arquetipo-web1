### Ejemplo de uso:

Componente:

- Para incorporar el component, el prefijo a utilizar para la etiqueta HTML es:

```html live=true
<mln-chart-circle
  [title]="title"
  [subText]="subText"
  [settingsChart]="settingsChart"
  [settingsButton]="settingsButton"
  [settingsCheckBox]="settingsCheckBox"
></mln-chart-circle>
```

Definición de modelo de datos del componente:

- title (string): Título del chart.
- subText (string): Texto que aparece debajo del botón 'SIGUIENTE'.
- settingsChart (Object):
